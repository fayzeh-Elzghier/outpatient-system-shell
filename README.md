# Outpatient System in Shell

A Linux-based outpatient management system implemented using Shell scripting.

## Features
- Register patients
- Book appointments
- Cancel appointments
- View appointments
- Simple menu-driven interface

## Technologies
- Bash / Shell scripting
- Linux terminal
- Text-based file handling

## Project Structure
- `scripts/` for shell scripts
- `data/` for patient and appointment records

## Notes
This project was developed as part of practical Linux/Shell scripting work.
